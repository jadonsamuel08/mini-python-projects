# Temperature Converter

A menu-driven CLI utility for converting temperatures between Celsius and Fahrenheit.

## Features

- Celsius to Fahrenheit conversion
- Fahrenheit to Celsius conversion
- Menu loop with invalid-option handling

## Concepts Practiced

- Function-based decomposition
- Numeric input and output formatting
- Interactive command-line menus and control flow

## Run

From the repository root:

```bash
python temp_converter/temp_converter.py
```
