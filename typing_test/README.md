# Typing Speed Test

A CLI typing test that measures speed and accuracy using randomly selected words.

## Features

- Generates a 30-word test sequence
- Countdown before each run
- Calculates:
  - Raw WPM (all typed words)
  - True WPM (correct words only)
  - Accuracy percentage
- Supports repeated attempts

## Concepts Practiced

- Timing execution with `time.time`
- Word-level comparison logic
- List comprehensions and summary metrics
- Randomized test generation

## Run

From the repository root:

```bash
python typing_test/typing_test.py
```
