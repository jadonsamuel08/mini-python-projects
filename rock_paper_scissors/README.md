# Rock Paper Scissors

A terminal-based Rock Paper Scissors game against the computer.

## Features

- User input with validation (`r`, `p`, `s`)
- Computer move generated randomly
- Round intro countdown for game feel
- Match result messages and replay loop

## Concepts Practiced

- Conditional logic and branching
- `match`/`case` control flow
- Random choices and basic timing (`time.sleep`)

## Run

From the repository root:

```bash
python rock_paper_scissors/rock_paper_scissors.py
```
